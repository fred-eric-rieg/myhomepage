Slack Clone "DA Bubble"

TypeScript | Angular 16 | SCSS | Material | Firestore Realtime Database | RxJS | Custom Pipes | Firestore Authentication | CI-CD Github Actions

Create channels, write text and post pictures. Also, reply in threads or write direct messages to other users. Edit your user profile and update your user profile picture. You can authenticate via your email or login with an anonymous guest account. Thanks to Firestore Realtime Database, reading and writing data happens with low latency - comparable to WebSockets.
