JOIN

Vanilla JS | HTML | CSS | PWA | Django | restAPI | Tokens

Full Stack Progressive Web App

JOIN is a Kanban Management Tool. Create, Read, Update and Delete Tasks. Authentication with Username & Password and Tokens.The frontend and the backend are connected via restAPI. The Frontend is currently hosted on my FTP server. The Backend is currently hosted on pythonanywhere.com. A Guestlogin is provided.
