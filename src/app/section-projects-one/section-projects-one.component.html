JavaScript | HTML | CSS | PWA

A Progressive Web App and task manager inspired by the Kanban system. Create and organize tasks using drag and drop functions, assign users and categories. Client-side login-authentication and password generation with SHA256 encryption and salt for demonstration purposes only. Mini backend (without security layer) by Junus Ergin.
A Guest Login is provided. The easiest way to download this PWA is via the Chrome browser on mobile or laptop.
